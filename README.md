# Сервис доставки

### В рамках [курса Кирилла Ветчинкина (11.09.2025 - 23.10.2025)](https://microarch.ru/courses/ddd/languages/java) реализуется один из сервисов Интернет-магазина.

### Были реализованы тактические паттерны DDD: 

- Aggregate, 
- Entity, 
- Value Object, 
- Domain Service, 
- Domain Event

- Docker
